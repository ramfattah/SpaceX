What is this?

Unofficial SpaceX Open Source REST API for rocket, core, capsule, pad, and launch data.

Who are you?

Why did you build this?

I like technology and cosmology.

Technical stuff?

I am using Django 3.0 Framework to display the front-end.

Base URL

The most current version of the API is v3, with the following base URL https://api.spacexdata.com/v3

Can I contribute to the project?

Of course, the API is provided by SpaceX and you can contribute to this project on GitHub.
